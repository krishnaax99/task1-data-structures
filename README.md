# Task 1 - Data Structures Implementation in C++

## Overview
This project implements two data structures from scratch:

1. Linked List
2. Binary Search Tree (BST)

The following operations are implemented:

- Insert
- Delete
- Search
- Display

## Time Complexity

### Linked List
- Insert: O(1)
- Search: O(n)
- Delete: O(n)

### Binary Search Tree (BST)
- Insert: O(log n)
- Search: O(log n)
- Delete: O(log n)

## How to Run

### Compile
```bash
g++ main.cpp -o app
```

### Run
```bash
./app
```

## Features
- Driver menu program
- Insert/Delete/Search operations
- Complexity comments included
